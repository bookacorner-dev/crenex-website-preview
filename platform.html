CRM compatibility

Already using Salesforce? Keep it where it fits.

Crenex and Salesforce solve different parts of the operating model. They do not need to duplicate each other.

01

Does Crenex replace Salesforce?

Not necessarily. Salesforce is a broad CRM for customer and sales relationships. Crenex supports the specialized operating workflow behind flexible leasing: spaces, availability, bookings, agreements, billing context and portfolio performance.

02

How does compatibility work?

The systems can be scoped to exchange agreed data while each remains responsible for the work it handles best. Exact behavior depends on your Salesforce edition, API access, data model, security requirements and the integration agreed during technical discovery.
